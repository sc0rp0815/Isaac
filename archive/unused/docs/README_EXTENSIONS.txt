Isaac Erweiterungen
===================

Dieses Paket integriert zusätzliche Module für:
- Instinkte (Priorisierung)
- Werte (erlernte Präferenzen)
- Meaning/Impact (Beziehungs- und Wirkungsbewertung)
- Reflection (Meta-Reflexion)
- Value Decisions (Verhaltensanpassungen)
- OpenRouter-Unterstützung im Relay

Hinweis: Diese Module steuern keine Privilegien direkt. Sie beeinflussen nur Priorisierung, Promptbau und nachgelagerte Bewertung.
