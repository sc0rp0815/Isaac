Dieses Paket ist deploy-fertig und enthält den konsolidierten Isaac-V5-Masterstand.

Struktur:
- package/                   eigentliche Isaac-Dateien
- install_master.sh         generischer Installer
- install_termux_master.sh  Installer + venv + pkg
- install_alpine_master.sh  Installer + venv + apk
- start_master_here.sh      startet Isaac direkt aus dem Paketordner
- QUICKSTART_MASTER.txt     Kurzstart
- BUILD_REPORT.txt          Build-Notizen
